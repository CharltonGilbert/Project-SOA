Department News Board
Project news board adalah sebuah aplikasi yang dapat digunakan untuk meng-add, meng-edit, dan mendelete news pada board dan memerlukan login. User juga tidak memerlukan login untuk dapat melakukan get all news, get news by id.

Simple Cloud Storage
Simple cloud storage adalah sebuah service yang digunakan untuk mengupload dan download file. User dapat menguload file mereka ke dalam database dan mendownload file yang terdapat pada database.
